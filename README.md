
## npm下载
```
npm install --save @heavy_leaf/my-vue-ui 
```

## 在项目中使用
```
// 导入包
import MyVueUI from '@heavy_leaf/my-vue-ui'
// 注册插件
Vue.use(MyVueUI)
```
